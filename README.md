# Scene Organiser

An OBS Studio dock plugin that organises scenes into colour-coded folders with drag-and-drop, live search, separators, and text annotations.

## Features

- Nested folders with colour bars
- Drag-and-drop reordering
- Live search
- Separators and text fields
- Per-scene-collection JSON persistence
- Bold highlight for active scene
- Native OBS scene context menu access

## Requirements

- OBS Studio 30.x / 31.x / 32.x
- Windows x64
- Qt 6 (bundled with OBS)

## Installation

Extract the release archive into your OBS Studio directory.
